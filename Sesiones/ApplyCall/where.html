Where

Expected input: where(spec)

Creates a new array that contains all the elements that satisfies the given specification.

spec is actually a callback function that will decide if the element shall or not to be included in the new list. 

      
        var people = [ 
          {name: 'pedro', age: 19 },
          {name: 'juan', age: 15 },
          {name: 'pablo', age: 16 },
          {name: 'pancho', age: 20 },
          {name: 'topo', age: 18 }
        ];
